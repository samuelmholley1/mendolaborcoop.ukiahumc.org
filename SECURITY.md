# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in the Mendo Labor Cooperative website, please report it responsibly:

1. **Email:** sam@samuelholley.com
2. **Subject:** [Security] Mendo Labor Coop Website Vulnerability
3. **Include:**
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Any suggested fixes (optional)

### What to Expect

- **Response Time:** We aim to acknowledge reports within 48 hours
- **Updates:** We'll provide updates on our progress
- **Credit:** We're happy to credit researchers who report vulnerabilities responsibly
- **Disclosure:** Please allow us reasonable time to address issues before public disclosure

### Out of Scope

- Issues affecting outdated browsers (we support modern browsers only)
- Social engineering attacks
- Physical security of member devices

## Security Measures

Our website implements:
- **HTTPS/TLS encryption** for all connections
- **Security headers** (CSP, HSTS, X-Frame-Options, etc.)
- **Input validation** and sanitization
- **No password storage** (we use URL-based access for internal pages)
- **Regular dependency updates**
- **Minimal attack surface** (static site with no database)

Thank you for helping keep our community safe!
